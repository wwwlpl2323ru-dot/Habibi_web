# Habibi — подключение формы к Google Таблице

Лендинг отправляет заявки в Google Таблицу через Google Apps Script (без бэкенда).

## Шаги

### 1. Создайте Google Таблицу
1. Откройте [Google Sheets](https://sheets.google.com) → создайте новую таблицу.
2. Назовите её, например, **Habibi Leads**.
3. Лист с данными создастся автоматически (скрипт сам создаст лист `Leads` с заголовками при первой заявке) — ничего вручную добавлять не нужно.

### 2. Добавьте Apps Script
1. В таблице откройте **Расширения → Apps Script**.
2. Удалите содержимое файла `Code.gs`, который открывается по умолчанию.
3. Скопируйте туда содержимое файла [`Code.gs`](Code.gs) из этого проекта.
4. Сохраните проект (значок дискеты, любое имя).

### 3. Опубликуйте как веб-приложение
1. В Apps Script нажмите **Развернуть → Новое развёртывание** (Deploy → New deployment).
2. Тип развёртывания — **Веб-приложение** (Web app).
3. Настройки:
   - **Execute as**: *Me* (ваш аккаунт)
   - **Who has access**: *Anyone* (Все, в т.ч. анонимные)
4. Нажмите **Развернуть** и разрешите доступ (потребуется подтвердить права для аккаунта Google).
5. Скопируйте полученный **URL веб-приложения** — он выглядит так:
   ```
   https://script.google.com/macros/s/XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX/exec
   ```

### 4. Вставьте URL в сайт
Откройте `index.html`, найдите строку:

```js
const ENDPOINT = ""; // e.g. "https://script.google.com/macros/s/XXXX/exec"
```

и вставьте туда скопированный URL:

```js
const ENDPOINT = "https://script.google.com/macros/s/XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX/exec";
```

### 5. Проверка
1. Откройте `index.html` в браузере.
2. Заполните и отправьте форму.
3. В Google Таблице должна появиться новая строка на листе **Leads** с колонками:
   `Date | Name | Contact | Interest | Language`.

## Обновление скрипта
Если позже измените `Code.gs`, нужно создать **новое развёртывание** (Deploy → Manage deployments → Edit → New version) — иначе изменения не применятся к уже выданному URL.
